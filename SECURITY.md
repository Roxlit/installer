# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Roxlit, please report it responsibly.

**Email:** security@roxlit.dev

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge your report within 48 hours and work on a fix as soon as possible.

## Scope

This policy covers:
- The Roxlit desktop app (this repository)
- The Roxlit Studio plugin

## Out of Scope

- Roblox Studio or Roblox platform vulnerabilities (report to Roblox directly)
- Rojo vulnerabilities (report to the Rojo project)
